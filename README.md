# Alchemic AI Landing Page

This folder contains a minimal GitHub Pages landing page for **alchemic.ai**.

## Files
- `index.html` – single-page site (logo + tagline)
- `logo.png` – web-friendly logo
- `favicon.ico` – browser tab icon
- `CNAME` – custom domain for GitHub Pages (`alchemic.ai`)

## Deploy
Commit these files to the root of your GitHub Pages repository, then enable GitHub Pages in repo settings.
